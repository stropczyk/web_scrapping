About the app

This is a portfolio project.

WebScrappingApp is an elegant and simple Python web application that will help you find best price for a product. This app is searching through database of one of the biggest store in Poland: komputronik.pl. All you need to do is just type product you're searching for in the Search box and save your money.

The following application uses:

→ Flask library

→ Beautiful Soup library

→ Bootstrap

→ Docker Compose
